# Koinz Task 

## How to run in dev mode ?

- First install dependencies by running (Be sure to have node v14+)
    ```
    npm install
    ```
- Run project in dev mode by running
    ```
    npm start
    ```
- Run test suites by running
    ```
    npm test
    ```
- Run E2E tests
    ```
    npm run cypress:open
    ```
